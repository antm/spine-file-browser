# UX Improvements Backlog

Suggested during code review and QA sessions. Pick these up in future sessions.

## Pending

### 1. Empty state messaging
When a folder has no files, show a helpful hint instead of a blank panel.
Example: *"Drag files here or use the ⋮ menu"*

### 2. Keyboard shortcut to focus the file list
A hotkey (e.g. `Mod+Shift+F`) to jump keyboard focus into the Spine file list without requiring a mouse click.

### 3. Scroll-to-focused item
When arrow-key navigating, ensure the focused item scrolls into view if it's off-screen.

### 4. Search clear button
An × button inside the search input to clear the query with one click rather than selecting all and deleting.

### 5. Folder collapse state persistence
Remember which subfolders are expanded/collapsed between sessions. Currently resets on every refresh.
